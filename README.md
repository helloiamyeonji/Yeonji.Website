# 个人简介网站模板

这是一个最简单的静态个人简介网站，适合直接上传到 GitHub Pages。

## 修改内容

- 在 `index.html` 里替换姓名、简介、项目、邮箱和链接。
- 如果想换头像，把自己的图片放到这个文件夹，并把 `index.html` 里的 `avatar.svg` 改成你的图片文件名。

## 发布到 GitHub Pages

1. 新建一个 GitHub 仓库。
2. 上传这些文件。
3. 打开仓库的 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main` 和 `/ (root)`。
5. 保存后等待几分钟，GitHub 会生成网站链接。
